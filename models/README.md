# 📢 Vosk 語音模型（請手動下載）

## 🔹 為什麼這裡是空的？
這個資料夾**不會被提交到 GitHub**，因為 Vosk 語音模型太大了。
請手動下載模型，並放入 `models/vosk_model/`。

## 🔹 如何下載？
1. 下載 **[vosk-model-small-cn-0.22](https://alphacephei.com/vosk/models)**（或更大的模型）
2. 解壓縮後，把資料夾改名為 `vosk_model`
3. 把 `vosk_model` 放到這個 `models/` 目錄內

🚀 **下載完後，你的專案應該長這樣：
voice_control_fish/ 
│── models/ 
│  ├── vosk_model/ # 這裡放模型檔案 
│ 
├── README.md # 這個文件
